# Contributing to MangaVault

First off, thank you for considering contributing to MangaVault! It's people like you that make this project awesome.

## 🚀 How Can I Contribute?

### Reporting Bugs
- Use the [GitHub Issues](https://github.com/walterwhite-69/MangaVault/issues) tracker.
- Describe the bug and include steps to reproduce it.

### Suggesting Enhancements
- Open a new issue with a clear title and description.
- Explain why this enhancement would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code follows the project's style (run a linter if possible).
5. Open a Pull Request!

## 📝 Coding Standards
- Keep code clean and efficient.
- Use descriptive variable names.
- Avoid unnecessary comments in production code.
- Ensure all new scrapers include proper error handling and anti-bot headers.
